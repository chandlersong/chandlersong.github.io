基础的数学，关于函数的知识

思维导图

函数

基本定义

一个对象转换成另一个对象的规则。 起始对象称为输入。定义域的集合返回独享是输出。

数学学多了，脑子里面都是数字，公式，但是函数，看到这句话，脑子里面还是眼前一亮的感觉。

上域

值域是上域的一个子集。 值域是实际输出的集合。 上域是可能输出的集合。

上域还是第一次听说，就当成是一个概念吧。

唯一的输出

一个函数必须给每一个有效的输入指定唯一的输出

区间

闭区间：[]
开区间：()

垂线检验

其实定义无非是网上抄一遍。没有什么太大的含义。
满足垂直校验的，就是函数。水平呢，就是可以求反函数

这里最大的感触，还是在于函数是图像的表达，而不是只有公式的表达。可能是从头到尾都在希望我们抛弃函数，只是公式的定式思维吧。

反函数

记做. \[f^{-1}\]

给定一个函数f,仅有一个x值满足f(x) = y才可能有反函数

有4个条件

  1. 从一个函数f出发，使得对于f值域中的任意y，都有为一个的x值满足f(x)=y, 也就是说，不同的输入对应不同那个的输出。
  2. \(f^-1\) 的定义域和f的值域相同
  3. \(f^{-1}\) 的值域和f的定义域相同
  4. \(f^{-1}(y)\)的值就满足f(x) = y的x。

复合函数

记做 \(f(x) = h(g(x))\) 或者 \(f = h \circ g\)

其实就是函数里面套函数。
g(x)的自变量是x
函数h的自变量，是g(x)。
f的自变量是x

奇偶性

偶函数

  1. f(x) = f(-x)
  2. y轴对称

奇函数

  1. f(x) = - f(x)
  2. 原点对称

特点

  1. 大部分函数都是飞奇非偶函数。

常用函数

线性函数

不同的表现形式

  1. \(y= mx + b\) m是斜率
  2. 点斜式： \(y-y_0 = m(x-x_0)\) 其中m是斜率
    • 通过两点\((x_1,y_1)\)和\((x_2,y_2)\)，那么它的斜率就是\(\frac{y_2 - y_1 }{x_2 - x_1}\)

三角函数

三角函数，主要用的弧度，也就是\(\pi\)系统

三角函数定义

数学表示

\[ \sin{\theta} = \frac{a}{c} \] \[ \cos{\theta} = \frac{b}{c} \] \[ \tan{\theta} = \frac{a}{b} \] \[ \csc(\theta) = \frac{1}{\sin{\theta}} \] \[ \sec(\theta) = \frac{1}{\cos{\theta}} \] \[ \cot(\theta) = \frac{1}{\cos{\tan}} \]

  • a 对边
  • b 邻边
  • c 斜边
  • csc 余割
  • sec 正割
  • cot 余切

坐标表示

\[ \sin{\theta} = \frac{y}{r} \] \[ \cos{\theta} = \frac{x}{r} \] \[ \tan{\theta} = \frac{y}{x} \]

r是到原点的距离。\(r = \sqrt{x^2+y^2}\)

常用三角函数的值

函数 0 \(\frac{\pi}{6}\) \(\frac{\pi}{4}\) \(\frac{\pi}{3}\) \(\frac{\pi}{2}\)
sina 0 \(\frac{1}{2}\) \(\frac{1}{\sqrt{2}}\) \(\frac{\sqrt{3}}{2}\) 1
cos 1 \(\frac{\sqrt{3}}{2}\) \(\frac{1}{\sqrt{2}}\) \(\frac{1}{2}\) 0
tan 0 \(\frac{1}{\sqrt{3}}\) 1 \(\sqrt{3}\) not exist

常用等式

\[ \tan{(x)} = \frac{\sin{(x)}}{\cos{(x)}} \] \[ \cot{(x)} = \frac{\cos{(x)}}{\sin{(x)}} \] \[ \sin^2{(x)} + \cos^2{(x)} =1 \] \[ 1 + \tan^2{(x)} = \sec^2{(x)} \] \[ \cot^2{(x)} + 1 = \csc^2{(x)} \] \[ \sin{(x)} = cos{(\frac{\pi}{2}-x)} \] \[ \tan{(x)} = cot{(\frac{\pi}{2}-x)} \] \[ \sec{(x)} = csc{(\frac{\pi}{2}-x)} \] \[ \sin{(A+B)} = \sin{(A)}\cos{(B)}+\cos{(A)}\sin{(B)} \] \[ \cos{(A+B)} = \cos{(A)}\cos{(B)}+\sin{(A)}\sin{(B)} \] \[ \sin{(2x)} = 2\sin{(x)}\cos{(x)} \] \[ \cos{(2x)} = \cos^2(x)- \sin^2(x) = 2\cos^2(x) - 1 = 1- 2\sin^2(x) = \cos^2(x)- 1 - \cos^2(x) = 1- \sin^2(x) - \sin^2(x) \]
